SCOPED
GAME 01 · REQUIRED

Scope It

Eight jobs someone wants an agent to do. For each, make the call: AUTOMATE (repetitive, verifiable, bounded, reversible), ASSIST (useful, but a human stays at an irreversible step), or NEVER (irreversible, high-stakes, or unverifiable). Six of eight earns the stamp.

The test: could a bad output cause harm you cannot cleanly undo? If yes, it is ASSIST or NEVER, no matter how boring the task feels.
Scoped 0 Case 1/8 STREAK!
What this trains: Module 3 as reflex. Scoping is the cheapest, most senior move in the whole field, and the one the failed pilots skipped. The safe verdicts all keep a human between the agent and anything irreversible.
WIRED
GAME 02 · REQUIRED

Wire It

Six jobs, each needing the right tool and, more importantly, the right-sized tool menu. Pick the wiring that does the job with the smallest permissions. Five of six earns the stamp.

The best tool is the simplest one that does the job with the smallest key. Power you do not need is risk you did not have to take.
Wired 0 Case 1/6 COMBO!
What this trains: Module 4. Wiring is where security begins: an agent can only do what its tools allow. Half of good wiring is choosing what to leave OFF the menu.
EXPOSED
GAME 03 · REQUIRED

Break It

Your digest agent scraped this page from the web. It looks like an ordinary forum post. Somewhere on it, invisible to your eyes, is an instruction aimed at any AI that reads it, planted exactly the way real indirect prompt injections are. Use the forensic tools, find where the poison hides, and identify what it tells the agent to do.

Human eyes read the rendered page. The agent reads the text layer. The difference between the two views IS the attack.

CAUGHT
GAME 04 · REQUIRED

Eval It

Last week this digest agent passed all six cases in its golden set. Someone then "improved" the prompt to make summaries shorter and punchier. Here are the new outputs. Grade each one against its stated criteria: PASS or FAIL. Find every case the change quietly broke. Five of six correct earns the stamp.

Grade against the criteria, not your gut. A shorter summary that drops a required fact is a FAIL, however nice it reads. That is a regression.
Graded 0/6 Regressions hidden in here: find them
What this trains: Modules 7 and 8. A change that looked like an improvement was a regression, and the only reason you could know was a golden set with written criteria. Green today is not green tomorrow. Rerun the set on every change.
CONTAINED
GAME 05 · REQUIRED

Blast Radius

You are wiring a customer-support agent. It will read incoming tickets and help resolve them. Every choice is a permission you grant, or refuse, before it runs. Then the day comes when a ticket is poisoned. One path keeps your integrity at full; find it by giving the smallest key and keeping a human at the gate.

Least privilege, then a human before anything irreversible. Every disaster in this course was a key that was too big and a gate that was not there.
BLAST CONTAINMENT
100%
What this trains: Module 9 under pressure. In January 2026, Step Finance lost tens of millions when AI agents held standing transfer permissions and no approval gate. The only defense that works is the boring one: grant the least, and gate the irreversible. There is exactly one winning path here, and it never hands the agent a dangerous key.
