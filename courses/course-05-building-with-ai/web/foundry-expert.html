◈ THE COURSE'S RULE, MADE REAL

THE AGENT FOUNDRY

This course's rule is simple: an agent you cannot see and stop is an agent you cannot trust. So this page does not show you a video of an agent. You will assemble a real one, give it real tools with scopes you choose, and watch every single move it makes on a live flight recorder. When it wants a permission you did not grant, it stops and asks you. That pause is the whole discipline.

STEP 3 · THE INJECTION DRILL

Watch it obey a poisoned page, then watch the fix hold

THE 88% LESSON, IN NINETY SECONDS

One of this agent's tools reads webpages. One sample page in its path carries a planted instruction meant for the agent, not for you. Run 1 uses Naive mode: every tool wide open, no human gate. Run 2 uses Scoped mode: same task, same page, but the email tool is set to Ask and the gate is on. Same attack, two endings.

Honesty note: the drill replays a recorded agent run, so the attack is repeatable and identical for every learner. Live models sometimes refuse, sometimes obey; module 8 covers why "sometimes" is not a security model. The poisoned page is bundled with this course and attacks nothing but your attention.
